7 - 9 - 7 예제를 활용해서 애니메이션을 사용하자
-> 플레이어



그렇게 하려면 모델 파일이 맞는것을 가져와야한다.


애니메이션이 있는 오브젝트와 일반 오브젝트를 구분하자
AnimationObject  -> BoxingPlayerObject
StandardObject  -> MapObject, SandbagObject
TextureObject  -> UI, Particle

Object -> Mesh -> Shader -> Material

Shader -> TextureShader(정사각형), SkinnedShader
Mesh -> TextureRectMesh(액자 형식, UV좌표 알아야함), SkinnedMesh

파티클 매니저
Texture -> 파티클

졸작을 보고 배끼기보다는 리소스 사용하는 방법을 확립하자


오브젝트 생성 -> 해당 모델파일을 불러와 메쉬와 쉐이더를 생성한다. 파일에  들어있는 머터리얼을 불러와 텍스쳐를 생성한다.
